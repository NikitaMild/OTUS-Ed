# Python QA Data Practice

Репозиторий для занятия "Работа с тестовыми данными".

## Структура
- `data/` - Тестовые файлы (json, csv, xml)
- `src/` - Утилиты для работы с данными
- `tests/` - Примеры проверок

## Запуск
Убедитесь, что у вас установлен Python 3.8+

Запуск примеров:
```bash
python src/json_utils.py
python src/csv_utils.py
python src/xml_utils.py
python src/context_managers.py
python tests/test_practice.py
